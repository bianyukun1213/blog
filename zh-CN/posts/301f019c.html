本项目已停止维护。

直接替换网站 JavaScript 脚本，再也不用开 Fiddler 去实现各种功能。

扩展程序与 JavaScript 脚本分别更新。

项目链接

主要功能（排名不分先后）

  • 赋予内容控制权限，无需老师同意即可使用白板与 PPT、踢出互动学生与旁听学生[已优化，按 Del(ete) 键配置是否启用]
  • 踢出老师与助教
  • 变更他人角色为互动学生或旁听学生
  • 掉线伪装[已优化，不再需要切换至管理员（巡课人员），按 End 键配置是否启用]
  • 免踢出
  • 更改使用者角色（1.1 版更新）
  • 切换房间（1.1 版更新，弊端是不会自动更新房间名称和 Token）
  • 更改使用者昵称（1.3 版更新）
  • 轮巡、导出花名册时无视角色、房间号（脚本更新）
  • 反向侦查（脚本更新，按 Home 键，可制裁管理员，可选踢出或将其更改为互动学生）
  • 自动签到（脚本更新）
  • 强制登录（1.6 版更新，弊端是无法获得正确的房间 Token）
  • 一键查看 PPT 图片[脚本更新，按 Ins(sert) 键查看，避免了最新版本中学生无法随意切换 PPT 导致不能查看的问题]
  • 屏蔽教师、助教视频（按 PageDown(PgDn) 配置是否启用）

使用方法

  1. 下载 扩展程序文件 并解压缩备用。
  2. 在扩展程序页面（chrome://extensions/）启用“开发者模式”。
  3. 点击“加载已解压的扩展程序”，选择解压缩过后的目录。
  4. 启用亮眼助手。（需正确的启用口令）

完
