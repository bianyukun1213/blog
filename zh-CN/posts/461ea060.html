以下内容包含剧透。

这里引用一下我对于 S05E10 的感想，但关于 The Machine 将要获得权限的那部分其实是错误的：

“我早知道 Root 会死。我以为她会死得壮烈，比如放着 Fortune Days 在激烈交火中身中数枪而死。我没想到她死得这么轻易，轻易到我甚至都没有什么感觉（除了看到那张瞪直勾勾地瞪着眼睛的大白脸时惊了一下，其实也会感到些许的惋惜）。不过这就像 Medicine 中提到的，他们都是 Irrelevant。前几季的时候我并不喜欢 Root 这个角色，但她后来洗白了我就非常喜欢了。她始终相信着 The Machine 和造物主 Finch。The Machine 模拟 Root 的相似度高达 99.6%，因此可以说 The Machine 就是 Root，Root 就是 The Machine。Root 死了，但最终 The Machine 为自己选了 Root 的声音，让我们能够继续享受原汁原味的 Root 式吐槽，这也解释了 Root 之前说的话——只要 The Machine 还在，这些人就还没有离开。而 The Machine 选择了 Root 的声音其实也还有另一种解释：root 是 UNIX 系统中的超级用户，The Machine 选择 Root 正如 Finch 将要逐步开放 The Machine 的权限，The Machine 获得了权限（root），The Machine 成为了 Root。”

而对于整部剧，我觉得结局还是非常不错的，我方仅仅死了两人，比我所设想的团灭结局好太多了。

以我的标准来看，绝对算是一个 Good Ending 了，毕竟大团圆的俗套结局大家都看惯了。

可惜的是，Root 不曾得知她所信仰的 God 选择了她的形象。

另外，本剧中计算机相关的内容实在是经不起推敲，心里默念“Finch 牛逼！”就行了，不要仔细思考。

现实生活中可能并不存在超级人工智能（也不一定），但大规模监控项目可不只是虚构的。
